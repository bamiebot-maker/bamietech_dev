# Ibrahim Sobur Bamidele - Portfolio

🚀 Modern, responsive portfolio website built with React, TypeScript, TailwindCSS, and Framer Motion.

## 🌟 Features

- **Modern Design**: Dark theme with gradient accents
- **Fully Responsive**: Works on all devices
- **Smooth Animations**: Powered by Framer Motion
- **Live Chat**: Integrated Tawk.to live chat
- **Contact Form**: Formspree integration for email delivery
- **Professional Sections**: Hero, About, Projects, Contact

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **Build Tool**: Vite
- **Deployment**: Vercel

1. Clone the repository:
\\\ash

cd portfolio
\\\

2. Install dependencies:
\\\ash

\\\

3. Start development server:
\\\ash
npm run dev
\\\
